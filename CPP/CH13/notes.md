# 文本模式 和 二进制模式
- 文本文件：使用二进制编码（如ASCII）的字符表示文本。
- 二进制文件：文件中的二进制值代表机器语言代码或数值数据或图片或音乐编码。
# c提供两种访问文件的途径
## 二进制模式
- 程序可访问文件的每个字节
## 文本模式
- 程序所见内容和文本实际内容不同
- 把本地环境的行末尾或文件结尾映射为C模式。
	- 例如：c在旧式macos以文本读取，把\r转换成\n,以文本写入吧\n转换成\r
# 标准文件
- C程序会自动打开三个文件
	-	标准输入  
	键盘，他是getchar()、scanf()使用的文件
	- 标准输出
	显示屏，他是putchar(),puts(),printf()使用的文件。
	- 标准错误输出
